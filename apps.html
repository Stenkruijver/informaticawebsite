Applicaties

Applicatie is de voluit geschreven naam voor apps. Apps zijn ook wel toepassingen en horen in de toepassingslaag van het drielagenmodel. Apps zijn software die ge√Ønstalleerd kunnen worden via de Google play store (Android) of de App store (iOS). Een aantal voorbeelden van apps zijn social media (instagram en whatsapp) of buienradars. Software-interfaces zijn de communicatiemiddelen om de verschillende software met elkaar te kunnen laten communiceren. Verschillende voorbeelden van software-interfaces zijn API (Application Programming Interfaces) zoals Android en iOS. De meeste apps halen data op via internet en dat doen ze door middel van een web-API. Software kan alleen contact maken met hardware via de logische laag (het besturingssysteem). Event Handlers worden gebruikt om bepaalde Events uitgevoerd te laten worden door de gebruiker. Voorbeelden van Events zijn swipe, press en zoom. Bij een knop gebruik je het Event press, bij websites gebruik je het Event swipe en bij een foto gebruik je het Event zoom.
